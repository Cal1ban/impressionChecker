Please run from terminal using Python 2.7.xx

Usage syntax is:

python path/to/script/checker.py ARG1 ARG2

ARG1:
The first argument should be the path to your dataset in .CSV format

EXAMPLE: path/to/my/data/dataSet.csv

ARG2:
The second argument should be the path to which to write the new .CSV file

EXAMPLE: path/to/write/my/output.csv


Both arguments are required and must include both the path and file name.
